## 📌 About

This is my personal portfolio website built with pure **HTML, CSS, and JavaScript** — no frameworks needed! Features a dark green theme with smooth animations, custom cursor, and scroll reveal effects.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Dark Theme** | Elegant dark green aesthetic |
| ⚡ **Animations** | Scroll reveal + fade-in effects |
| 🖱️ **Custom Cursor** | Green dot cursor with ring |
| 📱 **Responsive** | Works on all screen sizes |
| 🚀 **Fast** | Pure HTML/CSS/JS — no frameworks |
| 🔗 **Live Links** | Direct links to deployed projects |

## 🌐 How to Enable Live Portfolio

**GitHub Pages se free hosting:**
1. Repo → **Settings**
2. Left side → **Pages**
3. Source → **Deploy from branch**
4. Branch → **main** → **/ (root)**
5. **Save** click karo
6. 2-3 minutes baad live ho jayega!

